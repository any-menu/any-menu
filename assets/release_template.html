Download tips (English)
  • Obsidian Plugin Version:
    • You can download the three files: main, manifest, and styles
    • Or it is recommended to install using the brat plugin
    • Community store download method is currently under review: https://github.com/obsidianmd/obsidian-releases/pull/8345
  • Windows App Version: Download the msi or nsis file
  • Linux/macOS App Version: Pending future development and adaptation
