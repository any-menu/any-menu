快捷搜索

快捷搜索

当你通过全局快捷键展开面板后，会直接聚焦到搜索框中

  • 搜索内容
    • 支持搜索自定义短语
    • 支持搜索自定义脚本 (命令)
    • (开发中) 支持配置多级菜单实现带筛选的搜索
  • 搜索方式
    • 默认是模糊搜索，支持搜索匹配度排序
    • 支持拼音和首字母拼音检索。
      可在设置中关闭，对应选项:
      • pinyin_index = true
      • pinyin_first_index = true
    • 如果你比较熟悉输入法的自定义短语的功能，想要更高效率更精准的搜索，你可以在设置中将改为前缀匹配，对应选项:
      • search_engine = "trie"
  • 其他选项
    • serach_limit
