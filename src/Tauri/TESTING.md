# 测试系统托盘功能 (Testing Tray Features)

## 构建和运行

### 在 Linux 系统上构建
```bash
# 安装必要的系统依赖
sudo apt update
sudo apt install -y libgtk-3-dev libwebkit2gtk-4.1-dev librsvg2-dev

# 构建项目
cd src/Tauri
pnpm install
pnpm tauri build
```

### 在 Windows/macOS 上构建
```bash
cd src/Tauri
pnpm install
pnpm tauri dev    # 开发模式
pnpm tauri build  # 生产构建
```

## 测试功能

### 1. 启动测试
- 运行应用后，主窗口应该默认隐藏
- 应用程序在后台运行

### 2. 显示/隐藏测试
- 按 `Ctrl+Space` 或 `Alt+A` 应显示窗口
- 再次按相同快捷键应隐藏窗口
- 窗口应出现在鼠标光标附近

### 3. 自动隐藏测试
- 显示窗口后，点击窗口外部应自动隐藏
- 窗口失去焦点应自动隐藏

### 4. 关闭行为测试
- 点击窗口关闭按钮（X）应隐藏窗口而非退出程序
- 程序应继续在后台运行

### 5. 快捷键测试
- `Ctrl+Alt+Q`: 应完全退出程序
- `Ctrl+Alt+H`: 应隐藏窗口
- `Ctrl+Alt+S`: 应显示窗口
- `Esc`: 在窗口激活时应隐藏窗口

### 6. UI 元素测试
- 点击退出按钮应显示确认对话框
- 确认后应完全退出程序
- Pin 按钮应切换窗口置顶状态

## 预期行为

### ✅ 正常工作的功能
- 窗口初始隐藏
- 关闭窗口隐藏到后台
- 全局快捷键显示/隐藏
- 失焦自动隐藏
- 完全退出功能

### 📋 用户体验
- 类似传统系统托盘应用的行为
- 无需真实系统托盘图标即可获得托盘体验
- 通过键盘快捷键完全控制应用

## 故障排除

### 全局快捷键不工作
- 检查系统是否授予应用全局快捷键权限
- 确认快捷键未被其他应用占用

### 窗口不显示
- 检查控制台错误信息
- 确认 `show_window` 命令正确调用

### 无法完全退出
- 使用 `Ctrl+Alt+Q` 快捷键
- 或通过系统任务管理器结束进程
