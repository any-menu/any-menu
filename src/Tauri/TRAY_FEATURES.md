# 系统托盘功能 (Tray-like Features)

本应用实现了类似系统托盘的功能，提供了便捷的窗口管理和后台运行体验。

## 功能特性

### 1. 后台运行
- 关闭主窗口不会退出程序，而是隐藏到后台
- 应用程序继续在后台运行，可通过快捷键重新调出

### 2. 快捷键操作
- `Ctrl+Space` 或 `Alt+A`: 显示/隐藏主窗口
- `Ctrl+Alt+Q`: 完全退出程序
- `Ctrl+Alt+H`: 隐藏窗口
- `Ctrl+Alt+S`: 显示窗口
- `Esc`: 隐藏当前窗口

### 3. 智能窗口管理
- 窗口失去焦点时自动隐藏
- 窗口显示时自动定位到鼠标光标附近
- 支持置顶功能 (Pin按钮)

### 4. 用户界面
- 退出按钮：点击可完全关闭程序
- 置顶按钮：保持窗口始终在最前面
- 托盘操作说明：界面内显示快捷键提示

## 技术实现

### Rust 后端 (src-tauri/src/lib.rs)
- `show_window()`: 显示主窗口
- `hide_window()`: 隐藏主窗口
- `toggle_window()`: 切换窗口显示状态
- `quit_app()`: 完全退出应用程序

### 前端 JavaScript (src/window.ts)
- 全局快捷键注册和处理
- 窗口事件监听 (失焦自动隐藏)
- 点击穿透逻辑
- 与后端命令的集成

### 配置文件
- `tauri.conf.json`: 窗口初始配置为隐藏状态
- `capabilities/default.json`: 包含窗口管理权限

## 使用方法

1. **启动应用**: 程序启动后窗口默认隐藏
2. **显示窗口**: 使用 `Ctrl+Space` 或 `Alt+A` 显示窗口
3. **隐藏窗口**: 点击窗口外部、按 `Esc` 键或失去焦点时自动隐藏
4. **退出程序**: 使用 `Ctrl+Alt+Q` 或点击界面上的退出按钮

## 优势

相比传统的系统托盘实现：
- 无需复杂的系统依赖 (GTK, libappindicator等)
- 跨平台兼容性更好
- 实现更简单，维护成本更低
- 通过快捷键提供完整的托盘式体验

## 注意事项

- 确保全局快捷键不与其他应用冲突
- 在某些系统中可能需要授予应用全局快捷键权限
- 关闭窗口时程序仍在后台运行，使用快捷键或任务管理器完全退出
