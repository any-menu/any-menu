# panel 文件夹

目录

- (面板容器)
  - panelManager  | 管理多个主面板
  - index         | 管理多个子面板
- (子面板类)       | 又分核心子面板和脚本自定义子面板。脚本可以控制和监听前者
  - search/       | 搜索栏
  - toolbar/      | 工具栏
  - contextmenu/  | 多级菜单
  - pin/          | 置顶和拖拽按钮
- (共享类)
  - PanelItem     | 面板项
  - initTool      | 初始化主面板中的子面板

层级

- 面板容器 (同一个窗口只会存在一个)
  多窗口情况下 (使用拆分功能) 会存在多个，但只有一个是主面板
  - 子面板 (同一个窗口中的多个功能区域)
    - (可选) 面板项 (一般是按钮、功能按键)
